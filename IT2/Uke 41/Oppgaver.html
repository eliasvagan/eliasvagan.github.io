Det kan være hensiktsmessig å bruke funksjoner når du skal gjøre én ting mange ganger.

Oppgave 1

var P1 = document.getElementById("P1");

Definer P1 fra det elementet med ID-en "P1".

function navnet() {
P1.innerHTML += "Jeg heter Elias.";
}

Funksjonen "navnet" sier at P1 skal legge til teksten "Jeg heter Elias".

navnet();
navnet();
navnet();

Vi utfører funksjonen "navnet" tre ganger.

Oppgave 2

var P2 = document.getElementById("P2");

Definer P2 som variabel til elementet med ID "P2" (en p-tag)

var B2 = document.getElementById("B2");

Definer B2 som variabel til elementet med ID "B2" (en knapp)

P2.innerHTML = "Dette er en p-tag med tekst i seg. Denne teksten er skrevet i JS.
Lorem ipsum dolor sit "amet, consectetur adipisicing elit. Animi ea, hic id nemo neque officia omnis praesentium, quam quibusdam "quod sed vitae? Dolores excepturi labore possimus saepe, sint suscipit ut?";

P2 sit innhold er "Dette er en p-tag med [ ... ]".

B2.onclick = function() {
P2.style = `
background-color: lightblue;
font-size: 1.4em;
` }

Når man trykker på B2, endres stilen på P2 med følgende informasjon:

- Bakgrunnsfargen blir blå

- Font-størrelsen blir 1,4em.

Oppgave 3

var P3 = document.getElementById("P3");

Variabelen P3 er elementet på siden med ID-en "P3" (en p-tag).

var x = 1;

Variabelen "x" er tallet 1.

var sum = 0;

Variabelen "sum" er tallet 0.

while(x <= 100) {

sum += x;
x++;
}

Så lenge x er mindre/lik 100, fortsetter de følgende tingene å skje:

- Variabelen "sum" er lik seg selv pluss "x".

- Variabelen "x" øker sin egen verdi med én.
